# 🚀 Бэклог идей для TesseraDesk (от ИИ-критика)

## Запланированные исправления (Улучшение текущих функций)

1. **Встроенная мини-рисовалка (Избавление от mspaint.exe)**
   - *Зачем:* Сделать приложение полностью независимым от Windows (для будущей поддержки Linux/Mac).
   - *Идея:* Превратить текущий канвас редактора скриншотов в полноценную встроенную рисовалку внутри нашего "стеклянного" интерфейса.

2. **Быстрый скриншот без открытия редактора**
   - *Зачем:* Ускорить процесс, когда нужно просто скопировать область экрана.
   - *Идея:* Добавить в настройки галочку "Быстрый скриншот" (или отдельный хоткей), чтобы выделенная область сразу улетала в буфер обмена без открытия большого окна редактора.

3. **Открепляемые виджеты (Продвинутый компактный режим)**
   - *Зачем:* Дать возможность использовать несколько инструментов одновременно.
   - *Идея:* Позволить "отрывать" инструменты (таймер, калькулятор) от главного окна, чтобы они висели на экране как независимые маленькие виджеты.

## Новые фичи (Идеи для будущих версий)

1. **Умный буфер обмена (История скопированного)**
   - *Идея:* Добавить инструмент, который по горячей клавише будет показывать список последних 10-20 скопированных текстов или картинок для быстрой вставки.

2. **Облачная синхронизация**
   - *Идея:* Настроить резервное копирование заметок и задач (например, на Google Drive), чтобы данные пользователя не потерялись при поломке ПК.

3. **"Игровой" режим (Фокус-мод)**
   - *Идея:* Приложение должно понимать, когда запущена полноэкранная игра или фильм, и автоматически "засыпать", отключая все уведомления (например, от таймера), чтобы не мешать.
