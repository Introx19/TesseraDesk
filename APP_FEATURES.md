# 📚 Архитектура и Функции TesseraDesk (База Знаний)

Этот файл создан для ИИ-агентов, чтобы они понимали, какие функции уже существуют в приложении, и не предлагали сделать то, что уже работает.

## 🛠️ Основные Инструменты
1. **Таймер / Pomodoro (`MiniTimer.tsx`)**
   - *Функции:* Запуск таймера, пауза, сброс. Есть "Компактный режим" (виджет-кружок). Звуковые уведомления (с настройкой громкости и выбором звука в настройках).
2. **Секундомер (`Stopwatch.tsx`)**
   - *Функции:* Отсчет времени, сохранение кругов (Laps).
3. **Задачи (`Tasks.tsx`)**
   - *Функции:* Простой To-Do лист (Добавить, отметить выполненным, удалить).
4. **Заметки (`Notes.tsx`)**
   - *Функции:* Текстовое поле для быстрых записей с автосохранением.
5. **Калькулятор (`Calculator.tsx`)**
   - *Функции:* Базовые вычисления, сохранение истории. Интегрирован с DLC Формулы/Интегралы.
6. **Напоминания (`Reminders.tsx`)**
   - *Функции:* Запланированные уведомления.

## 📸 Скриншоты и Изображения
- **Умный Скриншот (`ScreenshotSelect.tsx` / `ScreenshotPreview.tsx`)**
  - *Функции:* Выделение области экрана. Предпросмотр с возможностью скопировать в буфер обмена, сохранить как файл, или открыть в системном `mspaint.exe`.
  - *Опции в настройках:* "Мульти-скриншот" и "Быстрый скриншот" (автоматическое копирование в буфер без предпросмотра).

## 🧑‍💻 Инструменты Разработчика (DLC)
Расположены в `src/components/dlc/DevTools.tsx`. Вкладки:
1. **Текст и JSON:** Форматирование, минификация, Base64 (Encode/Decode).
2. **Менеджер Портов:** Поиск процессов по порту и их завершение (`taskkill`). *(Работает только на Windows)*.
3. **Regex (Регулярные выражения):** Тестирование регулярок с подсветкой совпадений.
4. **Цвета (Пипетка):** Определение цвета пикселя с помощью `EyeDropper API` и перевод в HEX/RGB/HSL.
5. **Сниппеты:** Сохранение кусков кода. Имеют кнопку Свернуть/Развернуть.

## ⚙️ Настройки (`Settings.tsx`)
1. **Кастомизация UI:** Выбор темы (Glassmorphism, Neumorphism), выбор акцентного цвета.
2. **Горячие Клавиши (Global Shortcuts):** Назначаемые хоткеи для скрытия/показа приложения, быстрого скриншота, пипетки. Можно включить/выключить глобальные хоткеи (Game Mode).
3. **Локализация:** Поддержка `ru` и `en` через кастомный `i18n`.
4. **Звук:** Выбор звука таймера (Колокольчик, Диджитал, Кастомный `.mp3` файл), настройка громкости.
5. **Система:** Автозапуск вместе с Windows, Автоматические обновления (`electron-updater`).

## 🛑 Ограничения и Известные проблемы
- **Linux:** На Linux пока не работает Менеджер Портов и Редактор Скриншотов (из-за привязки к `mspaint.exe`).
- **Сниппеты:** Сохраняются локально в `localStorage`.
- **Иконка таскбара:** Для сборки правильной иконки нужен скопированный `build/icon.ico`.

*Примечание для ИИ: При планировании новых фич всегда сверяйся с этим файлом, чтобы убедиться, что функция еще не реализована!*
